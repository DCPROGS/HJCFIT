ExactSurvivor¶

class dcprogs.likelihood.ExactSurvivor(*args)¶

Computes exact missed-event survivor function.

Parameters:
  • qmatrix – The Q-matrix for which to compute the exact likelihood.
  • double (float) – The maximum length of missed events.
D_af(*args)¶
